# QUANTIX Analytics Dashboard
## Project Overview

**QUANTIX Analytics** is a futuristic business intelligence dashboard designed to monitor and visualize key performance metrics in real time.

The platform provides an interactive interface for tracking:

- Revenue growth
- User activity
- Transaction performance
- System health monitoring
- AI-powered analytical assistance

-----------

## Project Purpose

The purpose of **QUANTIX Analytics** is to help businesses monitor operational performance through visual analytics and real-time reporting.

It allows users to:

- Analyze monthly business performance
- Monitor active users
- Track financial growth
- Observe transaction activity
- Access AI-generated insights
- Review system status metrics

-----------
 ## Features
1. ### Sidebar Navigation

Provides access to dashboard sections including:

- Dashboard Overview
- Analytics
- Reports
- User Management
- Settings

The sidebar includes a collapsible hidden menu feature for better responsiveness.

2. ### KPI Performance Cards

Displays key business metrics such as:

- Total Revenue
- Active Users
- Transactions Processed
- Growth Percentage

These cards include hover animations and glow effects for modern presentation.

3. ### Performance Chart

Visualizes monthly performance data using vertical bar analytics.

Months displayed:

- January
- February
- March
- April
- May

The chart helps users identify trends and business growth patterns.

4. ### Live Activity Feed

Displays real-time updates including:

- New user registrations
- Payment activity
- Revenue spikes
- Traffic alerts
- Administrative logins


5. ### System Monitoring

Tracks platform infrastructure health:

- Server Status
- Memory Usage
- Uptime Performance

6. ### Quantum AI Assistant

An integrated AI-powered analytics assistant located at the bottom of the sidebar.

Purpose:

- Detect anomalies
- Predict business trends
- Provide optimization suggestions
- Enhance decision-making processes
